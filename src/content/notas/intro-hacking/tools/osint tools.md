# Herramientas: OSINT y Reconocimiento Pasivo
**Descripción:** Recursos para obtener información sin interactuar directamente con el objetivo (huella digital).

### Herramientas mencionadas
- **Hunter.io:** Búsqueda de correos electrónicos por dominio.
- **Phonebook.cz:** Indexación de dominios, correos y URLs de forma pasiva.
- **Pimeyes:** Motor de búsqueda de reconocimiento facial.
- **DeHashed:** Base de datos de filtraciones (leaks) de credenciales.
- **Whatweb / Wappalyzer / Builtwith:** Identificadores de tecnologías web (CMS, DB, frameworks).
- **Sublist3r / CTFR:** Herramientas de Python para enumeración pasiva de subdominios mediante certificados SSL o motores de búsqueda.

### Google Dorking
- `site:<dominio>`: Filtra por sitio.
- `filetype:<ext>`: Filtra por extensión de archivo.
- `intext:<texto>`: Busca palabras clave dentro del contenido.
