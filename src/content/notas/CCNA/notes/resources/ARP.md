ADDRESS RESOLUTION PROTOCOL

se utiliza para descubrir la direccion de capa 2 (direccion MAC) de una direccion de capa 3 ya conocida (direccion IP)

consiste de 2 mensajes
- ARP request: se manda como broadcat = enviado a todos los hosts en la red
- ARP reply: se manda como unicast = enviado solo a un host (el que mando el request)
